# FreeBSD Git Weekly

This repository is used to create the FreeBSD Git Weekly reports, which provide
a categorized list of commits to the FreeBSD source tree.

Reports are published at:
[https://freebsd-git-weekly.tarsnap.net/](https://freebsd-git-weekly.tarsnap.net/)

Anybody looking at this is probably interested in:
- [projects/freebsd/classify.toml](projects/freebsd/classify.toml)
- [src/commits_periodical/classify.py](src/commits_periodical/classify.py)
